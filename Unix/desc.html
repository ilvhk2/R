Get Summary statistics

R has a lot of datasets, to use one of the dataset called faithful.

Use attach function to load it to memory >

attach(faithful)

get summary of statistic >

summary(eruptions) 

   Min. 1st Qu.  Median    Mean 3rd Qu.    Max.
  1.600   2.163   4.000   3.488   4.454   5.100
use fivenum function to get minimum, 1st quartile, median,mean, 3rd quartile, maximum >

fivenum(eruptions) 

[1] 1.6000 2.1585 4.0000 4.4585 5.1000
