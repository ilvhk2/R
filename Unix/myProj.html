Big Data, Data Analytics, Data Mining, Data Visualization - these are all names for Statistical Data Analysis, which has become popular in recent years.

There are many software used for statistical analysis, e.g. R, SAS, MATLAB, SPSS, and Stata to name a few.

R is a free-to-use, open source language for high-performance computing. It integrates computation, visualization, and programming in an easy-to-use environment.

 

Install R

You could download the package from http://www.r-project.org

OR

You could install epel-release and then install R. R is present in EPEL(Extra Packages for Enterprise Linux) repository.

R has a lot of dependencies, I found it is not easy to install from the source code.  So for my project I do the simple way

sudo yum install R

sudo yum install epel-release

After R is installed, you could make a directory for storing your work in Linux

mkdir mywork

use setwd() to set your working directory and getwd() to see the working directory.

 

Syntax of R is similar like unix|linux

R is case sensitive

R default prompt is >

R commands are separated either by ";" or newline.

If command is not complete at the end of a line R will give a different prompt, by default is +

Use vertical arrow keys to scroll forward and backward through a command history

 

To get help: use help function or question mark

>help(plot)

>?plot

q to quit help

 

To start R

Type R at prompt $R

>q() to quit R

 

R workspace is stored collection of objects

Everything in R is an object.

 

Load package with library() Load object into memory with attach()
