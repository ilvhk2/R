You may want to export data, to do that , user write.table function in R

> write.table(x,file="test", append=FALSE,quote=TRUE,sep=" ",eol="\n",na="NA",dec=".",row.names=TRUE,col.name=TRUE,qmethod=c("escape","double"),fileEncoding="")

 

Write test data to .csv file

> write.csv(test,"test.csv",row.names=TRUE)

if you don't want to the row names, you need to specify row.names=FALSE

 

Write test data to .rda file

> save(test,file = "myTestData.rda")
